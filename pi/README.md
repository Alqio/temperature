## Installation

Follow the instructions from here: https://github.com/ttu/ruuvitag-sensor/blob/master/install_guide_pi.md

Required packages:
requests==2.22.0
ruuvitag_sensor
python-dotenv

## Notes
Run with crontab `*/5 * * * *`.

`find_ruuvitags.py` is a script for finding the MAC of your ruuvitag.
